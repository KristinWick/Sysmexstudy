# Sysmexstudy
This repository has the scripts that have been used in the Sysmex study published in this paper:

Wickstrøm KE, Holten AR, Prebensen C, Köhn-Luque A, Vitelli V, Amundsen EK. Sysmex Cell Population Data for Diagnosing Infection in Patients With Suspected Sepsis in the Emergency Department. Int J Lab Hematol. 2025 Oct 3. doi: 10.1111/ijlh.70008. Epub ahead of print. PMID: 41042192.
